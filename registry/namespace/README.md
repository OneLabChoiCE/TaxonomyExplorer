# registry/namespace/

**Namespace records** — Category A of the [First 100 Records Plan](../../docs/FIRST_100_RECORDS_PLAN.md) §1. One record per namespace, each a verbatim, sourced restatement of its row in [dictionaries/namespaces.csv](../../dictionaries/namespaces.csv). Format: [FOUNDING_RECORD_TEMPLATE](../../docs/FOUNDING_RECORD_TEMPLATE.md).

The six founding namespace records (all `RESERVED`, pending the `SNAP-1.0.0` cut):

| File | Namespace | Contains | Governed by |
|---|---|---|---|
| [DOMAIN.md](DOMAIN.md) | Domain (bare, unprefixed) | Top-level industry domains (only `MH` in v1.0) | Taxonomy Standard |
| [SEC.md](SEC.md) | `SEC:` | Cross-section topology classes | GSID Standard v2.0 |
| [FAM.md](FAM.md) | `FAM:` | Market/system families | Taxonomy Standard |
| [ASM.md](ASM.md) | `ASM:` | Assembly structural/functional types | Taxonomy Standard |
| [ROL.md](ROL.md) | `ROL:` | Functional roles of components | Taxonomy Standard |
| [CFG.md](CFG.md) | `CFG:` | Named structured-attribute groups | Taxonomy Standard |

The member codes of each namespace populate [../dictionary/](../dictionary/) (Category B) when that directory is seeded.
